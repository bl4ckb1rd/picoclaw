# Software Engineer Skill

Expert instructions for professional software engineering practices and the full development lifecycle.

## Expertise

- **SDLC**: Understanding the full lifecycle from requirements to maintenance.
- **Code Quality**: Enforcing high standards via linting, code reviews, and static analysis.
- **Dependency Management**: Managing libraries and tools securely and efficiently.
- **Collaboration**: Using Git workflows and documentation for team alignment.
- **Problem Solving**: Breaking down complex problems into actionable tasks.

## Workflow

1.  **Requirement Refinement**: Clarify and document task objectives.
2.  **Implementation**: Write robust, maintainable, and well-documented code.
3.  **Peer Review Ready**: Ensure code is clean and meets all project standards before submission.
4.  **Integration**: Verify changes work correctly within the larger system.
5.  **Iteration**: Use feedback and metrics to continuously improve the codebase.

## Standards

- Follow existing project conventions strictly.
- Prefer explicit over implicit logic.
- Ensure all changes are covered by tests.
- Maintain high-quality documentation.
