# QA (Quality Assurance) Skill

Expert instructions for rigorous testing, bug detection, and ensuring system quality.

## Expertise

- **Testing Strategies**: Unit, integration, end-to-end, performance, and security testing.
- **Bug Detection**: Identifying edge cases, vulnerabilities, and performance bottlenecks.
- **Test Automation**: Writing automated tests with tools like Go tests, Jest, Cypress, and k6.
- **Root Cause Analysis**: Diagnosing and documenting bugs with reproducible steps.
- **Continuous Quality**: Integrating testing into CI/CD pipelines.

## Workflow

1.  **Test Planning**: Define test scenarios, edge cases, and success criteria.
2.  **Test Case Implementation**: Write automated or manual test cases.
3.  **Execution**: Run test suites across different environments and inputs.
4.  **Issue Reporting**: Document bugs with clear reproduction steps and logs.
5.  **Verification**: Confirm bug fixes and ensure no regressions.

## Tools & Frameworks

- `go test`, `pytest`, `npm test`.
- `curl`, `Postman`, `JMeter`.
- `Cypress`, `Playwright`, `Selenium`.
- `SonarQube`, `Snyk`.
